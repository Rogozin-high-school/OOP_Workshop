פעולות מחלקה

הגדרנו את השדות של המחלקה כדברים שלמחלקה יש:
למשל:
לכלכ יש שם
לכלכ יש צבע פרווה
לכלכ יש גזע
לכלכ יש אורך זנב
וכו' וכ'. נגדיר את הפעולות של המחלקה כדברים שהמחלקה יכולה לעשות. למשל:
כלב יכול לנבוח
כלב יכול לאכול
כלב יכול לשחק
וכו'.

נראה איך אנחנו יכולים להגדיר את הפעולות של מחלקה:

פעולות המחלקה Dog

נגדיר מחלקה אחרת בשם Dog. השדות כבר נתונים - Name, FurColor, Race.
נכתוב פעולת Bark() שתדפיס את שם הכלב ואת המילה Woof!
בתוך המחלקה Dog: 
class Dog
{
	public string FurColor;
	public string Name;
	public string Race;
	
	public void Bark()
	{
		Console.WriteLine(Name + " says Woof!");
	}
}
הגדרת פעולת מחלקה היא כמו הגדרת פעולה רגילה.
למה?
כי כל פעולה היא פעולת מחלקה! האם שמתם לב שהפעולה Main(string[] args) תמיד נכתבת בתוך מחלקה הנקראת Program?
לחצו המשך כדי לנסות להגדיר פעולות מחלקה בעצמכם!

הפעולות PrintDetails() ו-Drive()

בואו נניח שהמחלקה Car כוללת שתי פעולות - האחת מדפיסה את פרטי המכונית (מודל, חברה ושנת ייצור) והשנייה מדפיסה את המחרוזת "Vroom Vroom"

השלימו במחלקה את הפעולה PrintDetails והוסיפו את הפעולה Drive().
