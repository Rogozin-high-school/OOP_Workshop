שלב 1

קצת מונחים:
מחלקה - התיאור של הטיפוס - מה יש בתוכו, מה הוא עושה, וכו'
שדה - כל משתנה המוגדר בתוך המחלקה הוא שדה של המחלקה. למשל - במחלקה Student, השדות הם Grade ו-Name.
תכונה - שם נרדף לשדה. לרוב משתמשים במונח האנגלי Property.

יכול להיות שעלתה השאלה, מה המילה public עושה?
המילה public לפני הגדרת השדה נועדה לציין שהשדה הוא ציבורי - ז"א, שניתן לגשת אליו מחוץ למחלקה. מה זאת אומרת?

עכשיו ננסה ליצור משתנה מהטיפוס שיצרנו, ואז נראה איך משתמשים בו.

יצירת עצמים

כמו שראינו בשלב הקודם, ניתן להגדיר עצם מטיפוס מסוים באותה צורה שמגדירים כל משתנה רגיל מטיפוס רגיל.
* עצם - הכינוי למשתנה מטיפוס שהוגדר על ידי מחלקה. למשל - משתנה מסוג Student הוא עצם, אבל משתנה מסוג int הוא לא.
ניצור עצם מהטיפוס Car שבנינו בשלב הקודם: 
public static void Main(string[] args)
{
	Car myCar;
}
כעת, האובייקט myCar מכיל את הערך null. הערך null נועד לציין חוסר ערך - שהאובייקט לא מכיל כלום.
נאתחל את האובייקט למופע חדש של המחלקה Car.
* מופע - instance - בעוד שהמחלקה מגדירה רק אילו שדות/ערכים יש לטיפוס הנתונים שלנו, מופע של טיפוס הנתונים הוא עצם ממשי שמחזיק ערכים ממשיים.
דוגמא לכך: המחלקה "מורה" תכיל שני שדות - שם וצבע תיק (לדוגמא). זוהי המחלקה.
יש מורה ששמה דיאנה וצבע התיק שלה הוא חום. זהו המופע - אובייקט ספציפי של הטיפוס "מורה".

נאתחל את האובייקט myCar למופע חדש של המחלקה Car: 
public static void Main(string[] args)
{
	Car myCar = new Car();
}
המילה השמורה new תיצור אובייקטים/מופעים חדשים של מחלקות שונות בC#.

השמת ערכים בתוך השדות

עד עכשיו אתחלנו את האובייקט myCar. אבל מה עם השדות שבתוכו? עלינו לאתחל גם אותם!
ניתן לפנות אל החברים* של המחלקה בעזרת שימוש בנקודה. דוגמה: 
public static void Main(string[] args)
{
	Car myCar = new Car();
	myCar.Model = "Model S";
	myCar.Company = "Tesla";
}
בדוגמה זו אתחלנו את השדות Model ו-Company בערכים Model S ו-Tesla, בהתאמה. בגלל שלא אתחלנו את השדה השלישי, הוא יישאר עם הערך ברירת מחדל של הטיפוס int, שהוא 0.
* חבר - כרגע ההתייחסות לשדות, בהמשך גם לפעולות המחלקה. חברי המחלקה הם השדות והפעולות שלה.
נסו בעצמכם ליצור מופע חדש של המחלקה Car, לחצו על המשך כדי לתרגל.
בתוך פונקציית הMain, צרו משתנה מסוג Car באיזה שם שתבחרו. אתחלו את השדות Model ו-Company למה שתבחרו, ובשדה ProductionYear רשמו 2005.
הדפיסו את השנה ואת שם המודל של המכונית בעזרת Console.WriteLine().
